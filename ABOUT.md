1. Обзор проекта
Проект представляет собой инструмент Dynamic Application Security Testing (DAST) — систему автоматизированного поиска уязвимостей в веб-приложениях. Система выполняет краулинг целевого приложения, генерирует специально сформированные нагрузки и анализирует реакцию сервера для обнаружения уязвимостей класса OWASP Top 10.

Охват уязвимостей (OWASP Top 10)
SQL-инъекции • XSS (Cross-Site Scripting) • SSRF (Server-Side Request Forgery) • Открытые редиректы • Небезопасная конфигурация HTTP-заголовков • Прочие классы согласно OWASP Top 10

3. Архитектура системы
3.1 Компонентная диаграмма

Верхний уровень
CLI / Web UI / REST API   └─ Orchestrator (управление сканом, пауза/возобновление)        ├─ Auth Manager (OAuth, Basic Auth, Cookie, Bearer)        ├─ Crawler (граф приложения, ссылки, формы, JS-маршруты)        ├─ Payload Engine (генерация нагрузок из словарей)        ├─ Analyzers: Signature | Heuristic        ├─ State Store (PostgreSQL — граф, очередь, результаты)        └─ Report Generator (PDF + JSON)

3.2 Описание компонентов

Компонент	Ответственность
Crawler	Обход страниц, извлечение ссылок, форм, JS-роутов, API-эндпоинтов. Построение графа приложения.
Auth Manager	Сессионные cookie, HTTP Basic Auth, OAuth Bearer-токены, автоматическое прохождение форм логина.
Payload Engine	Генерация нагрузок для GET/POST-параметров, заголовков, cookie из встроенных и пользовательских словарей.
Signature Analyzer	Поиск в теле ответа и заголовках маркеров эксплуатации (сигнатурный анализ).
Heuristic Analyzer	Сравнение базового и аномального ответов (статус-код, тело, время отклика).
Orchestrator	Управление глубиной обхода, таймаутами, паузой/возобновлением, областью сканирования.
Report Generator	Формирование отчёта: классификация уязвимостей, доказательная база, рекомендации, экспорт PDF/JSON.
State Store	PostgreSQL: граф URL, очередь задач, найденные уязвимости, состояние сессии сканирования.
REST API + Web UI	Управление сканами, просмотр результатов, администрирование пользователей, загрузка словарей.

10.2 Структура docker-compose.yml

Сервисы Docker Compose

Воркер → Crawler → Auth Manager → Payload Engine → Analyzers → PostgreSQL
